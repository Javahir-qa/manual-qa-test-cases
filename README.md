# Manual QA Project

### Tool
- Qase.io

### Project Description
Manual testing of a web application to ensure core functionality works as expected.

### Test Scope
- Login functionality
- Balance top-up
- Contact form
- Header and footer functionality

### Test Types
- Functional testing
- Negative testing
- Boundary value testing

### Deliverables
- Test cases documented in Qase.io
- Bug reports (if any) with steps to reproduce
- Test execution results
